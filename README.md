### STATISTICAL ARBITRATION - MIMUW 2025

The course focuses on data-driven methods for analyzing financial market 
inefficiencies. It introduces students to fundamental economic concepts and 
theories related to the valuation of financial instruments.

Topics covered include techniques for aggregating diverse data types 
(both numerical and textual) from multiple sources and analyzing them within 
a cloud environment. 
This environment is also utilized to build tools for automating asset 
management strategies, including those based on machine learning. 
The performance of the proposed solutions will be evaluated against 
real-world benchmarks, such as stock market indices.

### Used tools
 - pandas, numpy, matplotlib
 - IBM Cloud
 - Keras, Tensorflow, XGBoost, CatBoost
